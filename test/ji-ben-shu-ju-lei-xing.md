9种基本数据类型，以及他们的封装类

| 数据类型 | 字节 | 位 | 取值范围 | 封装类 |
| :--- | :--- | :--- | :--- | :--- |
| byte | 1 | 8 | -2^7 ~ 2^7-1 | Byte |
| short | 2 | 16 | -2^15 ~ 2^15-1 | Short |
| int | 4 | 32 | -2^31 ~ 2^31-1 | Integer |
| long | 8 | 64 | -2^63 ~ 2^63-1 | Long |
| float | 4 | 32 |  | Float |
| double | 8 | 64 |  | Double |
| char | 2 | 16 | \u0000 ~ \uffff | Character |
| boolean |  |  |  |  |

* 浮点数默认类型为double
* char类型是单一的十六位Unicode字符
